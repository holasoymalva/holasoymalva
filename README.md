# My Hacker Profile 🚀

## 💻 Graphical View (Recommended)

> [!WARNING]
> **Do NOT copy and paste the raw SVG XML code directly into your README.md file.** GitHub's markdown sanitizer does not support inline SVG styling, which causes the inner CSS code and raw text nodes to display below the card as garbage text. Instead, download the `profile.svg` file, commit it to your repository, and link it as shown below.
